# コンテンツ運用ガイドライン（抜粋）

## 自社サイトと note サテライトの運用ルール

- **公開順序**
  - まず IYASAKA 公式サイト（`/iyasaka/articles/...` や `/iyasaka/products/...`）側にフルコンテンツを公開する。
  - note への要約公開は、**自社公開から48〜72時間後**を目安とし、「先にnoteだけ公開」は行わない。

- **canonical の指し先**
  - すべてのnote記事は、**canonical を自社サイト（公式ドメイン）の該当URLに向ける**。
  - 自社サイト側はあくまで「正本」、noteはサテライト・要約という位置づけとする。

- **note側コンテンツの考え方**
  - 要約・導入・骨子レベルに留め、詳細な手順・チェックリスト・事例の深掘りは自社サイトに誘導する。
  - 直接的なコピー＆ペーストではなく、**要約・再編集・視点の変化**を加えること。

このルールにより、「検索・AIからの引用の起点は常に自社サイト」としつつ、noteを安全にトラフィックブースターとして活用する。
