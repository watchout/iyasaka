# MBR（月次経営レビュー）

## 目的

- 月次で「仮説の更新」と「重点の入替」を行い、戦略を微修正する
- 財務（キャッシュ）とパイプラインを同時に見て、速度と安全性を両立する

## 役割（最小）

- 進行（Facilitator）: 時間管理、結論の確定
- 記録（Scribe）: 月次スコアカードと決定事項の更新
- 数字オーナー: 財務/主要KPIの集計責任

## 事前準備（必須）

- `04_scorecards/MONTHLY_SCORECARD.md` を更新する
- `03_kpi/KPI_TREE.md` の前提（率/必要量）を実測で更新する

## タイムボックス（推奨 60分）

1. 主要KPIの総括（15分）: 何が増えたか/減ったか
2. パイプライン（15分）: 次月の見込み、詰まりポイント
3. 財務（15分）: 入金・固定費・現預金、当面のリスク
4. 学びの抽出（10分）: 勝ち筋/負け筋を言語化
5. 次月の重点（5分）: 最大3つ、成功条件をKPIで固定
