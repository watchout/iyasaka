# 実行ログ（施策・学びの蓄積）

## 目的

- 「何をやったか」を残し、勝ち筋/負け筋を早く見つける
- 週次WBRで、施策→結果→学び→次の一手が回る状態にする

## 記入ルール（最小）

- 1施策=1行でよい（詳細は必要になったら足す）
- 施策は **仮説 → 実行 → 結果 → 学び → 次** をセットで書く

## ログテンプレ（追記式）

| 日付 | チャネル | 施策 | 仮説 | 期待KPI | 結果 | 学び | 次の一手 |
|---|---|---|---|---|---|---|---|
| YYYY-MM-DD | 紹介/提携/アウトバウンド/サイト/コンテンツ |  |  |  |  |  |  |
