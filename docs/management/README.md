# 経営ディレクトリ（SSOT）

このディレクトリは、IYASAKAの **経営計画・事業計画・KPI運用** を「常に蓄積し続ける」ための正本（Single Source of Truth）です。

## 目的

- **3ヶ月で「ホテルAI：無料先行予約 3件」** を達成するために、目標→分解→実行→レビュー→改善を週次・月次で回す
- 施策・意思決定・学びが散らからず、後から見返して再現できる状態にする
- 現場視点（不→光→弥栄）と数値運用（KPI/PL/キャッシュ）を両立する

## 既存ドキュメントとの役割分担

- **ブランド/ナラティブ（世界観の正本）**: `docs/ssot/*`
- **実装/運用（開発・サイト運用の正本）**: `docs/ops/*`
- **経営運用（KPI/会議体/パイプラインの正本）**: `docs/management/*`（このディレクトリ）

## 更新ルール（最小で回る）

- **週次**: `04_scorecards/WEEKLY_SCORECARD.md` と `07_execution/EXECUTION_LOG.md` を更新し、`05_meetings/WBR_WEEKLY.md` に沿ってレビューする
- **月次**: `04_scorecards/MONTHLY_SCORECARD.md` と `05_meetings/MBR_MONTHLY.md` を更新し、重点KPI・戦略仮説を更新する
- **用語/定義が揉めたら**: `03_kpi/KPI_DICTIONARY.md` を更新して統一する（口頭で決めない）
