# IYASAKA Website Production Document

> CursorAIなどのAIエディタを使ってHPを制作する際、AIに「設計図」兼「指示書」として読み込ませるためのシステム・プロンプト用テキスト。
> `.cursorrules`に設定するか、チャットの冒頭で読み込ませることで、**デザイン、コピー、コードのすべてにIYASAKAのナラティブが貫通したサイト**を構築できる。

---

## 1. ブランド定義（コンセプト・OS）

CursorAIには、まずこの「世界観のルール」を理解させます。

```markdown
# Brand Concept: IYASAKA（弥栄）
- Mission: 社会と企業の「不（痛み）」を、仕組みと現場力で「未来の光（希望）」へ変え、ともに弥栄する循環をつくる。
- Narrative OS (The 3 Pillars):
  1. 整える (Organize): 現場の不透明・属人化をインフラと設計で基盤化する（弱電、ミエルプラス）
  2. つなぐ (Connect): 分断された情報、言語、体験を接続し拡張する（配信プラス、多言語プラス）
  3. 育てる (Nurture): AI伴走やデータの活用で、永続的な繁栄の循環を作る（OmotenasuAI、AIプラス、農業）
- Key Tone: 誠実、先進的、現場主義、体温のあるテクノロジー
```

## 2. サイト構成（Sitemap）

AIに生成させるページの階層構造です。

```markdown
# Site Structure
- Top Page: 「不」から「光」へのストーリーテリング。3つの提供価値の提示。
- Services & Products (Category):
    - [整える] 弱電プラス、ミエルプラス（Board / Stock / Drive / FILE）
    - [つなぐ] 配信プラス、多言語プラス
    - [育てる] OmotenasuAI、AIプラス
- About Us: 社名「IYASAKA」に込めた想い、未来の展望（農業・循環）
- Case Studies: 現場がどう変わったかの物語
- Contact: 共創の始まり
```

## 3. 各製品のナラティブ・データ（コンテンツ集）

AIが各ページの内容を書くためのソースデータです。

| カテゴリ | 製品名 | 不（Before） | 光（After/Bright） | 弥栄（Benefit） |
| --- | --- | --- | --- | --- |
| **整える** | **弱電プラス** | 施工ミス、保守の不安、専門外の悩み | 24/365の安心、確かな技術基盤 | 現場のストレス解消、本業への集中 |
| **整える** | **ミエルプラス** | 在庫不明、予定の属人化、管理漏れ | 全情報の可視化、ミスの自動検知 | チームの信頼向上、時間の創出 |
| **つなぐ** | **配信プラス** | 配信のハードル、リソース不足 | どこでも配信、プロ品質の常駐化 | 場所とスキルの新しい価値化 |
| **つなぐ** | **多言語プラス** | 言語の壁、再撮影のコスト | 既存動画のグローバル化、AI翻訳 | 世界中へのファン拡大 |
| **育てる** | **OmotenasuAI** | フロントの多忙、データ分断 | パーソナライズされた接客、自動化 | 顧客満足度の向上、スタッフの余裕 |
| **育てる** | **AIプラス** | ツールの形骸化、使いこなせない | 業務への定着、独自AIの進化 | 企業の持続的な成長、DXの完遂 |

## 4. CursorAI への具体的指示（プロンプト例）

CursorAIのチャット欄に以下のように指示してください。

### A. トップページのヒーローセクション作成時

> 「ブランド定義」を元に、トップページのヒーローセクションを実装して。
> コピーは「今ある『不』を、未来の『光』へ。」をメインにし、IYASAKAの3つの動詞（整える・つなぐ・育てる）を象徴する視覚的なカードレイアウトを作成して。農業のビジョンはフッター付近に控えめに予感させる程度で。

### B. 製品詳細ページの作成時

> 「製品名：ミエルプラス」の詳細ページを作成して。
> 構成は「不→原因→仕組み→光→弥栄」のナラティブ・ループに従うこと。単なる機能紹介ではなく、ホワイトボードの管理で疲弊している現場監督が、解放されて笑顔になるようなストーリーをベースにしたテキストを生成して。

### C. デザインのトンマナ指示

> 全体的に「清潔感のある白」と「信頼感のある紺」を基調にしつつ、アクセントカラーには「夜明けの光」をイメージした暖かいオレンジ（#FF9E00系）を使って。硬いテック企業ではなく、現場に寄り添う温かみのあるモダンなUIを目指して。

## 5. 特記事項：農業セクションの扱い（AIへの制約）

農業については、AIが勝手に「現在の事業」として盛り込みすぎないよう、以下の制約を与えます。

```markdown
# Constraint for Agriculture Section:
- 農業については、現在のサービスラインナップには含めないこと。
- 「未来の展望」や「IYASAKAが目指す究極の循環」として、AboutページやVisionセクションで触れるのみに留める。
- 言葉選びは「命の循環」「次世代への還元」など、精神的な深みを持たせること。
```
